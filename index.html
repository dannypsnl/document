Why llir/llvm?

The target of llir/llvm is: interact in Go with LLVM IR without binding with LLVM. Therefore, you don't have to compile LLVM(could take few hours), no fighting with cgo. Working under pure Go environment and start your journey.

Installation

To install llir/llvm, all you need to do is: go get github.com/llir/llvm.

Usage

According to packages, llir/llvm can be separated to two main parts:

  1. asm: This package implements a parser for LLVM IR assembly files. Users can use it for analyzing LLVM IR files.
  2. ir: This package declares the types used to represent LLVM IR modules. Users can use it for build LLVM IR modules and operating on them.
